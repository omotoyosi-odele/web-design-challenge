Summary: Latitude vs X

This website showcases a weather visualisation project that aims to determine the variations in weather as one gets closer to the equator where latitude is 0, hence the emphasis on latitude. Using the OpenWeatherMap API and several Python libraries like matplotlib, pandas, scipy and some external libraries like citipy, we analysed the weather data of over 500 cities across the world and plotted several charts showing different elements like temperature, humidity, cloudiness and wind speed to determine what variances exist across the latitudes, which divide the earth into northern and southern hemispheres. The data source used to run the analysis and the visualisations created, along with explanations of observed trends are also showcased.
